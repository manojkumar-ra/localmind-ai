# LocalMind AI

A private local AI chat application running 100% offline.

## Features
- 100% offline - no internet required
- No API key needed
- Data never leaves your device
- Model selection - Llama 3.2, Mistral
- Creativity level control
- Professional dark orange theme

## Tech Stack
- Python
- Streamlit
- Ollama
- Llama 3.2

## How to Run
1. Install Ollama from ollama.com
2. Run: ollama pull llama3.2
3. Install dependencies: pip install streamlit requests
4. Run: streamlit run ollama_app.py
